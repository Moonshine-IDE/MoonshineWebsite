Download Moonshine SDK Installer

What is Moonshine SDK Installer?

To take full advantage of Moonshine IDE you will need third party SDKs like Apache® Flex or Apache® Royale. In order to make it easier for you to setup the required SDKs, we created the Moonshine SDK Installer.


Moonshine IDE will download and install Moonshine SDK Installer automatically on Windows platform. If you're using macOS you will need to download and install Moonshine SDK Installer from the list below.
